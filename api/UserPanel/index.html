UserPanel

This class was designed to contain user relevant properties and methods used to customize the user features.

Functions​

SetIconRetrieveHandler​

</>
UserPanel.SetIconRetrieveHandler(
iconRetrieveHandler: (plr: Player) → string--

This function passes the player as an arg then should return a image id

) → ()

This function sets the iconRetrieveHandler which is a function that gets called when a users icon is being generated for their user frame.

note

This handler can be set to override a players icon in LeaderboardExtended.
